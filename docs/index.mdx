## Introduction

AI‑powered payment integration framework for AI agents. Automate financial workflows in your applications through a unified API that connects M‑Pesa, Airtel Money, Equity Bank, and KCB using the Model Context Protocol (MCP).

## What is PayLink?

PayLink is an open‑source framework that standardizes payment integrations for AI agents via MCP. It provides one consistent interface to multiple providers (M‑Pesa, Airtel Money, Equity Bank, KCB), so your applications can initiate and manage payments without bespoke, provider‑specific code.

## Key Features

- **Multi‑Provider Support**: M‑Pesa, Airtel Money, Equity Bank, KCB
- **AI‑First Design**: Built specifically for AI agents and MCP integration
- **Secure Authentication**: Enterprise‑grade security with OAuth support
- **Instant Setup**: Get payment processing running in minutes
- **East Africa Focus**: Specialized for regional payment systems

## Why PayLink for your agent

- Integrate once, access many: one API for M‑Pesa, Airtel Money, Equity Bank, and KCB
- Managed authentication: provider credentials handled securely with consistent flows
- Built on MCP: native to AI agents, reliable tool invocation and context control
- Production‑ready: sensible defaults, idempotency patterns, and error handling
- Faster time‑to‑value: ship payment features without bespoke provider code
